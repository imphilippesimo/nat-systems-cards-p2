# nat-system-cards-p2

##Prérequis:

* JDK 8+
* Maven 3.x.x
* Mysql 
   Database: cards
   Username: cards
   password: cards 


###Installer les dépendances:
Se placer à la racine du projet  
`mvn install`

###Tests:
Se placer à la racine du projet   
`mvn test`

###Execution:
Toujours à la racine du projet  
`mvn spring-boot:run`
